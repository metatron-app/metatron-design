Asset Parameter¶

Asset Detail의 Parameter tab에서는 Asset에서 수집하고 있는 전체 Sensor/Parameter 중에서 사용자가 관심이 있는 Parameter를 등록할 수 있습니다. 그리고 사용자가 등록한 Parameter들에 대해서 실시간 모니터링을 할 수 있는 기능을 제공합니다.

APM은 Asset의 건강도를 확인할 수 있는 다수의 알고리즘을 제공합니다. 위 그림에서는 Spec Rule, SPC Rule, Variation Detection 그리고 RUL 까지 총 4개의 알고리즘을 사용하여 Asset의 건강도를 확인하고 있습니다.

Asset Detail의 Parameter tab에서는 특정 기간 동안 Parameter 추이를 모니터링 할 수 있는 기능을 제공합니다. 사용자가 확인하고자 하는 기간을 선택하면 손쉽게 사용자가 등록한 Parameter들에 대한 이력데이터를 조회할 수 있습니다.
